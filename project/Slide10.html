|- Elements, Markup, and Tags -|

HTML is made up of Elements.

Elements(*) contain:
  • Text Content (What you're looking at)
  • Markup(*) (What it looks like)
  • Refs to other docs (e.g. images, videos)
  • Links to other pages (Other things to look at)
  • HTML Markup.


HTML Markup(*) consists of:

  • opening tag <tag name>
  • the tag contents
  • closing tag </tag name>



    • HTML elements are either 1) inline or 2) block.

    Block, as the name might suggest, makes an 'invisible box' around whatever is inside the element.
    This box has dimensions which can be exploited. (There will be more on this later.)
    An inline element occupies only the space reserved by the tags that define the inline element
